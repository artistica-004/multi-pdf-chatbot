# 📚 Multi-PDF Chatbot with Source Citations

Upload multiple PDFs and ask questions across all of them. Get answers with exact source citations showing which document and page was used.

## Features
- Upload multiple PDFs simultaneously
- Ask questions across all documents
- Source citations with PDF name and page number
- Powered by Groq LLaMA 3.3 70B
